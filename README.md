### Design rationale (why this reads as "premium")
- **One color, one style.** A single charcoal badge color with white logos + `flat-square` avoids the rainbow look recruiters read as filler.
- **Cohesive stat system.** All four cards use `555555` gray on transparent backgrounds with hidden borders, so they look designed rather than pasted in, and adapt to both GitHub light and dark modes.
- **Restraint.** Exactly four widgets (stats, top-langs, streak, activity) — current 2026 guidance warns that stacking many widgets makes a profile look noisy.
